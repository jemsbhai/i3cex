# I3C-EX

I3C-EX is an independent research content protocol for edge AI/ML metadata and
coordination over I3C Basic v1.2.

Start with the repository [README](https://github.com/jemsbhai/i3cex#readme),
the [current specification](https://github.com/jemsbhai/i3cex/blob/main/specs/I3CEX-0.2.0-draft.md),
and the [standards alignment policy](./standards/I3C_BASIC_V1_2_ALIGNMENT.md).

This project is not affiliated with, sponsored by, approved by, or endorsed by
MIPI Alliance. Passing project tests does not establish I3C Basic conformance.
